Live database

Activar guardado desde la página.

Esta fase deja la web preparada para leer y escribir recursos desde una base liviana. La integración pensada es con Supabase: vos completás la configuración, y después los recursos nuevos aparecen solos en la home y en cada sección.

Paso 1

Crear la tabla

Ejecutá este SQL en tu proyecto de Supabase para crear la tabla y las políticas base.

docs/database-setup.sql
Paso 2

Completar config.js

Activá la conexión en el archivo de config con tu URL y tu key pública del proyecto.

docs/config.js
Paso 3

Guardar desde la web

Desde el botón Agregar recurso, el formulario va a crear filas nuevas en la base y la web las va a volver a leer al cargar.

home + section views
Config

window.UXUI_DB_CONFIG = {
  enabled: true,
  provider: "supabase",
  supabaseUrl: "https://TU-PROYECTO.supabase.co",
  supabaseKey: "TU-ANON-KEY",
  table: "ux_resources",
  statusColumn: "status",
  publishedValue: "published",
};
