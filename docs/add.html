Live database

Activar guardado y uploads desde la página.

Esta fase deja la web preparada para leer y escribir recursos desde Supabase. Una vez conectado, vas a poder crear links, subir PDFs o archivos reales y verlos reflejados solos en la home y en cada sección.

Paso 1

Crear la tabla

Ejecutá este SQL en tu proyecto de Supabase para crear la tabla y las políticas base.

docs/database-setup.sql
Paso 2

Activar Storage

Corré el setup del bucket público y las políticas de upload para aceptar PDFs, documentos y assets.

docs/storage-setup.sql
Paso 3

Completar config.js

Activá la conexión en el archivo de config con tu URL, tu key pública y el bucket que va a guardar uploads.

docs/config.js
Paso 4

Guardar desde la web

Desde Agregar recurso, el formulario va a crear filas nuevas en la base y, si subís archivo, también lo va a cargar a Storage automáticamente.
